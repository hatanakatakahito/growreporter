# GROW REPORTER プラン仕様書

最終更新日: 2025年1月24日

## 📋 プラン体系概要

GROW REPORTERは、3つのプラン体系でサービスを提供します。

### プラン一覧

| プラン | 月額料金 | 想定ユーザー |
|--------|---------|-------------|
| 無料プラン | ¥0 | 個人ブロガー、スタートアップ、試用ユーザー |
| スタンダードプラン | ¥9,800（税込 ¥10,780） | 中小企業、複数サイト運営者、Web制作会社 |
| プレミアムプラン | ¥19,800（税込 ¥21,780） | 大企業、マーケティング代理店、EC事業者 |

### プラン制限のチェックロジック

```javascript
// サイト登録時のチェック
const canAddSite = (userPlan, currentSiteCount) => {
  const limits = getPlanInfo(userPlan).features;
  return currentSiteCount < limits.maxSites;
};

// AI生成時のチェック
const canUseAIFeature = (userPlan, featureType, currentCount) => {
  const limits = getPlanInfo(userPlan).features;
  const monthlyLimit = featureType === 'summary' 
    ? limits.aiSummaryMonthly 
    : limits.aiImprovementMonthly;
  
  // 999999 = 無制限
  return currentCount < monthlyLimit || monthlyLimit >= 999999;
};
```

## 📝 補足事項

### データ保持期間について
- すべてのプランで「無制限」としていますが、これはGA4の標準保持期間（14ヶ月）に準拠します
- Firestore側での追加的な制限は設けません

### AI生成回数の「無制限」について
- プレミアムプランの「無制限」は、実装上は 999,999 回/月として扱います
- 実質的に制限なく利用可能

### プラン変更について
- プラン変更は即時反映
- アップグレード：即座に新機能が利用可能
- ダウングレード：次回請求日から適用
- サイト数がプラン上限を超える場合は、超過サイトを選択して削除する必要あり
